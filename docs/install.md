# Install

## With NPM

You can install Movigo library with npm:

```bash
    npm i @movigo/core --save
```
    
## With CDN

You can also use CDNs:
    
```html
    <script src="https://unpkg.com/@movigo/core"></script> <!-- Unpkg -->
    <!-- or -->
    <script src="https://cdn.jsdelivr.net/npm/@movigo/core"></script> <!-- JSDelivr -->
```
